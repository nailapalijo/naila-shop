 Naila Shop - Premium Tech E-commerce
Naila Shop is a modern, high-performance, and fully responsive e-commerce landing page built for premium tech gadgets. The project focuses on a sleek user interface, smooth animations, and a functional client-side shopping experience.

 Key Features
Dynamic Cart System: Real-time cart updates (add/remove) powered by JavaScript and persistent localStorage.

Fully Responsive Design: Optimized for a seamless experience across Mobile, Tablet, and Desktop using Tailwind CSS.

Premium UI/UX: Features glassmorphism effects, custom floating animations, and modern typography.

Interactive Catalog: A dynamic product grid with categories and instant "Add to Cart" functionality.

Clean Codebase: Modular HTML structure with optimized JavaScript logic to prevent errors.
